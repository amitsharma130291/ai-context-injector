Getting started
Three steps, then it's automatic
You set it up once. Every AI session after that takes 2 seconds.
1
Pin the extension
Click the puzzle 🧩 icon in Chrome's toolbar → find AI Context Injector → click the pin icon. The ⚡ icon will now always be visible.
2
Create your snippets
Click ⚡ → click + New → write your project context (stack, conventions, current task). Use {{variables}} for things that change. Save. Done once.
3
Inject on any AI chat
Open ChatGPT, Claude, Gemini, or any supported AI → click ⚡ → click Inject. Your context appears in the chat input instantly. Then just hit Send.
Important to know
No copy-pasting. Ever.
The extension types directly into the AI chat input for you.
💉
How inject actually works
When you click Inject, the extension finds the chat input on the page and writes your snippet content directly into it — exactly like you typed it yourself, but instantly.

You do not need to copy anything. You do not need to paste anything. Just click Inject, then hit Send in the AI chat.

If you're on a tab that isn't an AI platform, the button says Copy instead — it copies the snippet to your clipboard as a fallback.
✓ No copy · No paste · One click
🔄
Injecting twice replaces, not appends
If you update a snippet and inject again, it replaces whatever was in the chat input — it does not append to it. So you can freely re-inject if you change your mind about which snippet to use.
Power feature
Variables — dynamic context in every snippet
Write {{variable}} anywhere in a snippet. At inject time, a modal asks you to fill them in.
Project: {{project}} → My SaaS App
Stack: {{stack}} → Next.js 14, TypeScript, Tailwind
Task: {{task}} → Fix the auth redirect bug
Date: {{date}} → 12 Jul 2026  (auto-filled)
Client: {{client}} → Acme Corp  (any custom name works)

💡 Tip: The {{task}} variable pre-fills from the ⚡ Quick Task bar — update your task for the day in one tap and every snippet that uses {{task}} will pick it up automatically.

Full feature list
Everything in v2.0
All features work locally. Nothing is sent to any server.
🔀Variables
Write {{anything}} in a snippet. On inject, a modal asks you to fill each one in. {{date}} fills automatically. Use any name you like.
📊Token counter
Every snippet card shows an estimated token count (~420t). Goes yellow above ~1,500 tokens, red above ~3,000 — so you never accidentally blow Claude's context window.
☑️Multi-select inject
Tick the checkbox on 2 or more snippets → a bar appears at the top → click Inject all. All selected snippets are combined into one block (separated by ---) and injected together.
🗑Multi-select delete
Select multiple snippets with checkboxes → click 🗑 Delete in the bar. Confirms before deleting. No need to go into the editor to delete one at a time.
📌Pin / Star snippets
Click the ☆ on any card to pin it. Pinned snippets always sort to the top of the list, above everything else — even above your most-used snippets.
📋Duplicate snippet
Click the 📋 icon on any card to clone it as "(copy)". Useful for creating variations of a snippet for different clients or projects without starting from scratch.
⌨️Keyboard shortcut
Alt+Shift+I opens the popup from anywhere in Chrome — no need to click the toolbar icon. Change it at chrome://extensions/shortcuts if needed.
⚡Quick task field
When you're on an AI tab, a yellow bar appears at the top of the popup: "Today's task:". Update it in one tap. It auto-fills the {{task}} variable across all your snippets.
🔝Auto-sort by most used
The snippets you inject most often automatically rise to the top of the list. You never have to scroll to find your go-to snippet — it's always first. Each card shows its use count (Nx).
🖱️Right-click inject
Right-click inside any AI chat input box → ⚡ Inject AI Context → pick from your top 10 snippets in the submenu. Inject without opening the popup at all.
📁Folders
Assign each snippet to a folder (Work, Freelance, Personal, General — or any custom name). Filter by folder using the dropdown in the toolbar. Snippets group visually when "All folders" is selected.
💾Import / Export JSON
Click 📤 to export all snippets to a .json file — backup, or send to a colleague. Click 📥 to import. Merges intelligently — no duplicate snippets created if you import the same file twice.
🔍Search
Search bar at the top filters snippets by title, content, tags, and folder — instantly as you type. Find any snippet in milliseconds even with 50+ saved.
📈Usage stats dashboard
Click 📈 to open the stats panel. Shows: total injects, estimated time saved, top snippets ranked by use (with bar chart), and usage broken down by folder.

Get the most out of it
Tips from day one
Things that make a real difference.
🗂️
Create one snippet per project, not per task. Put your stack, conventions, and repo structure in the snippet. Use {{task}} for what changes daily. One snippet covers the whole project.
📌
Pin your most-used snippet. If you have one project you work on every day, pin it. It'll always be at the top — zero scrolling, one click away.
☑️
Use multi-inject for layered context. One snippet for project context, one for coding rules, one for the specific bug you're debugging. Tick all three → inject → Claude gets the full picture in one shot.
🖱️
Use right-click for speed runs. Once you know your snippets well, right-click the AI input → pick the snippet → done. Never open the popup. Even faster than Alt+Shift+I.
📤
Export your snippets regularly. Takes 2 seconds. Your snippets live in Chrome storage — if you clear site data or switch to a new machine, you'll want that backup. Export to a folder you sync (Dropbox, Drive, iCloud).
📊
Watch the token counter. Claude has a ~200k token context window but ChatGPT-4o is tighter. If your snippet badge turns red (~3,000+ tokens), consider splitting it into two smaller snippets by topic.
⚡
Update Quick Task every morning. It takes 5 seconds — type what you're working on in the yellow bar. Every snippet using {{task}} picks it up automatically all day. No editing snippets needed.
